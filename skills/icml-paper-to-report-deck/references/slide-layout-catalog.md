# Slide Layout Catalog

## Available Layouts

### title
- Centered title (28–34 pt, Aptos Display)
- Subtitle line for author/venue
- Dark background variant allowed
- Optional visual asset centered below title

### assertion_figure
- Assertion title at top (28–32 pt)
- Large figure/table in main area (preserves aspect ratio)
- Takeaway line below figure
- Source footer at bottom (8–9 pt)

### two_column
- Assertion title at top
- Left column: 2–4 body lines (16–20 pt)
- Right column: figure or table
- Source footer at bottom

### method_flow
- Assertion title at top
- 3–5 sequential boxes/components
- Brief labels per component
- Source footer at bottom

### result_highlight
- Assertion title with metric name
- Large formatted numeric result
- Comparison bar or arrow
- 1–2 supporting detail lines
- Source footer at bottom

### comparison
- Assertion title
- Side-by-side comparison of 2–3 methods/results
- Bar or table visualization
- Source footer at bottom

### limitations
- Assertion title
- 3–5 limitation items with brief explanations
- Each item gets one line + small detail text
- Source footer at bottom

### takeaway
- "Key Takeaways" header
- 3 concise takeaway points
- Each with supporting evidence reference
- Dark background variant allowed

## Geometry Constants

```
Slide: 13.333" × 7.5" (16:9 widescreen)
Margins: 0.6" sides, 0.45" top
Footer: y=7.08", height=0.20"
Minimum inter-block spacing: 0.35"
```
